Angular JS is a JavaScript framework that can extend the capabilities of HTML with angular directives (called ng-directives). The three most important directives are ng-app, ng-model, and ng-bind. The ng-app directive is used to declare that an AngularJS application is being used. The ng-model directive is used to bind HTML control values to data and the ng-bind directive binds data to an HTML view. Directives make working with data straightforward. They can also be used to make fast changes to CSS and other page details. In this exploration, I examine how to take advantage of what Angular is capable of for a single page application.
